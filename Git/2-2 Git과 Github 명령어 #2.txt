restore의 역할 2가지.

1) Staged -> Untracked 로

$ echo "Git..." > Temp.md
$ git add .\Temp.md
$ git status

Changed .... // Staged 상태.

$ git restore --staged Temp.md  // Staged -> Untracked 로 변경하는 방법.
$ git status

Untracked  // Staged -> Untracked 변경된 것을 확인할 수 있다.

2) modified 에서 Staged 로. 이전 상태로 돌아가게 된다.
